# An-Agent-Based-Model-to-Explore-the-Adoption-of-Travel-Plans-Among-SMEs
Car miles driven must fall by 9% before 2035 to reach carbon budget targets. Approximately, 24% of all car miles driven are commutes. This agent based model explores how social dynamics (incl. norms, mimetic isomorphism, and strategic complementarity) could lead to an s-shaped curve of adoption among smaller firms.
